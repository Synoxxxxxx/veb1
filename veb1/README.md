# Лабораторная работа: FastCGI сервер для проверки попадания точки в область

## Описание проекта

Данный проект реализует FastCGI сервер на Java для проверки попадания точки на координатной плоскости в заданную область. Проект включает в себя:

- HTML страницу с формой для ввода координат
- CSS стили с использованием различных селекторов
- JavaScript валидацию и AJAX запросы
- FastCGI сервер на Java для обработки запросов
- Алгоритм проверки попадания точки в область

## Структура проекта

```
src/
├── main/
│   ├── java/org/example/
│   │   ├── FastCGIServer.java    # Основной FastCGI сервер

│   │   └── Main.java            # Главный класс для запуска
│   └── resources/
│       └── index.html            # HTML страница с формой
└── test/
```

## Алгоритм проверки области

Область состоит из трех частей:

1. **Прямоугольник в I квадранте**: `0 ≤ x ≤ R, 0 ≤ y ≤ R/2`
2. **Четверть круга во II квадранте**: `x² + y² ≤ (R/2)², x ≤ 0, y ≥ 0`
3. **Треугольник в III квадранте**: `x ≥ -R/2, y ≥ -R/2, x + y ≥ -R/2, x ≤ 0, y ≤ 0`

## Параметры ввода

- **X**: чекбоксы со значениями от -4 до 4
- **Y**: текстовое поле для ввода значения от -5 до 5
- **R**: радиокнопки со значениями 1, 1.5, 2, 2.5, 3

## Запуск проекта

### Предварительные требования

- Java 17 или выше
- Maven 3.6 или выше

### Команды для запуска

1. **Сборка проекта:**
   ```bash
   mvn clean compile
   ```

2. **Запуск FastCGI сервера:**
   ```bash
   mvn exec:java -Dexec.mainClass="org.example.FastCGIServer"
   ```

3. **Альтернативный запуск:**
   ```bash
   java -cp target/classes org.example.FastCGIServer
   ```

4. **Тестирование алгоритма:**
   ```bash
   mvn exec:java -Dexec.mainClass="org.example.PointChecker"
   ```

### Доступ к приложению

После запуска сервера откройте в браузере:
```
http://localhost:8080
```

## API

### Endpoint: `/cgi-bin/point-checker`

**Метод:** GET

**Параметры:**
- `x` - координата X (от -4 до 4)
- `y` - координата Y (от -5 до 5)  
- `r` - радиус области (1, 1.5, 2, 2.5, 3)

**Пример запроса:**
```
GET /cgi-bin/point-checker?x=1&y=0.5&r=2
```

**Пример ответа:**
```json
{
  "x": 1.0,
  "y": 0.5,
  "r": 2.0,
  "hit": true,
  "timestamp": "14:30:25",
  "executionTime": 5
}
```
